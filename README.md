# Управление портфелем: эффективные портфели на российских акциях

Проект посвящен применению теории Марковица на реальных данных 30 российских акций с Московской биржи за 2015-2025 годы. Построены границы эффективных портфелей (efficient frontier) при различных ограничениях на веса, рассчитаны бета-коэффициенты по рыночной модели, проведено сравнение трех подходов к оценке ковариационной матрицы (историческая выборочная, на основе рыночной модели, на основе скорректированных бета), проверена теорема двух фондов Блэка и построена граница методом Монте-Карло.

Выполнены задачи 1-24 из 25 (включая все бонусные). Не начата задача 25 (учет транзакционных издержек).


## Как запустить

```bash
pip install -r requirements.txt
```

Открыть `portfolio_analysis.ipynb` и выполнить ячейки по порядку. Все данные уже лежат в `data/` и ничего дополнительно скачивать не нужно. Сводный файл `data/export/data_export.xlsx` содержит все основные результаты на 39 листах.

Если нужно воспроизвести сбор данных с нуля (через MOEX ISS API и ЦБ РФ), скрипты загрузки лежат в `scripts/` (файлы 01-06). Ключи доступа не требуются, все источники публичные.

Зависимости: pandas, numpy, matplotlib, seaborn, scipy, statsmodels, pyarrow, openpyxl, apimoex, pytest. Полный список с версиями в `requirements.txt`.

### Границы эффективных портфелей (задачи 4-8)

Выбрано окно: скользящее 252 дня, последний торговый год, конец 30.12.2025.
Безрисковая ставка: 16% (ключевая ставка ЦБ на конец 2025).
Оптимизатор: scipy SLSQP, ftol = 1e-12.

| Режим ограничений | Доходность портфеля минимального риска (GMVP) | Волатильность GMVP | Коэффициент Шарпа |
|-------------------|-----------------------------------------------|---------------------|--------------------|
| Без ограничений | -9.4% | 13.6% | -1.86 |
| Короткие позиции до 25% | -9.4% | 13.6% | -1.86 |
| Только длинные позиции | 1.2% | 17.4% | -0.85 |
| Минимум 2% в каждую акцию | -0.2% | 20.6% | -0.78 |

Коэффициент Шарпа отрицателен во всех режимах. Это типичная ситуация для российского рынка 2025 года, когда ключевая ставка (16%) выше ожидаемой доходности GMVP.

Вложенность границ: чем жестче ограничения на веса, тем выше минимальный риск.

### Бета-коэффициенты (задачи 11-12)

- Индекс: IMOEX (полное покрытие 2015-2025, все 30 акций входят в состав)
- Окно: 252 дня, метод наименьших квадратов (OLS), конец 30.12.2025
- Бета от 0.46 до 1.42, среднее около 1.0, все значимы (p < 0.001)
- Скорректированная бета (формула Блюма): 2/3 * бета + 1/3 (сжатие к единице)

### Ковариационная матрица по рыночной модели (задачи 13-15)

Формула: Sigma = бета * бета' * дисперсия рынка + diag(остаточные дисперсии). Требует 61 параметр (30 бета + 30 остаточных дисперсий + 1 дисперсия рынка) вместо 465 у выборочной ковариации.

| Режим | Метод оценки | Доходность GMVP | Волатильность GMVP |
|-------|-------------|-----------------|---------------------|
| Без ограничений | Выборочная | -9.4% | 13.6% |
| Без ограничений | Рыночная модель | -8.5% | 13.0% |
| Только длинные | Выборочная | 1.2% | 17.4% |
| Только длинные | Рыночная модель | -0.1% | 17.2% |

Рыночная модель дает чуть меньший GMVP-риск за счет фильтрации шумовых корреляций. Число обусловленности 158.9 (ниже, чем 186.5 у выборочной).

### Ковариационная матрица на скорректированных бета (задачи 16-18)

Та же формула, но с бета по Блюму (2/3 * бета + 1/3). Остаточные дисперсии те же.

| Режим | Метод оценки | Доходность GMVP | Волатильность GMVP | Число обусловленности |
|-------|-------------|-----------------|---------------------|------------------------|
| Без ограничений | Выборочная | -9.4% | 13.6% | 186.5 |
| Без ограничений | Рыночная модель | -8.5% | 13.0% | 158.9 |
| Без ограничений | Скорректированная бета | -9.1% | 16.6% | 152.4 |

Число обусловленности у скорректированной бета наименьшее среди трех методов. При этом волатильность GMVP выше: сжатие бета к единице убирает экстремальные решения оптимизатора, но поднимает нижнюю границу волатильности. Скорректированная бета стабильнее во времени за счет регрессии к среднему.

### Теорема двух фондов Блэка (задача 22)

Проверена теорема Блэка (1972): любой портфель на границе без ограничений является линейной комбинацией любых двух других портфелей с этой же границы.

| Проверка | Точность | Вывод |
|----------|----------|-------|
| Числовая (GMVP + касательный, 200 портфелей) | max L2 = 1.2e-05 | Теорема подтверждена |
| 5 пар базисных портфелей | max L2 = 1.3e-05 | Работает для любой пары |
| Аналитическая (формулы Мертона, 1972) | примерно 8.7e-16 | Математически подтверждено |
| Только длинные (с ограничениями) | max L2 = 0.24 | Теорема не выполняется |
| Короткие до 25% (с ограничениями) | max L2 = 7.11 | Теорема не выполняется |

Практический смысл: в режиме без ограничений достаточно двух фондов (GMVP + касательный) для синтеза любого портфеля на границе. При наличии ограничений на веса линейная структура нарушается.

### Граница методом Монте-Карло (задача 23)

500 000 случайных портфелей для каждого из 3 режимов. Генерация весов: распределение Дирихле для длинных позиций, нормальное с нормализацией для неограниченных, отбраковка (rejection sampling) для ограниченных коротких. Граница извлекается как верхняя огибающая (200 бинов по доходности, минимальная волатильность в каждом).

| Режим | Волатильность GMVP (МК) | Волатильность GMVP (аналит.) | Зазор | Шарп касат. (МК) | Шарп касат. (аналит.) | Зазор |
|-------|--------------------------|------------------------------|-------|-------------------|------------------------|-------|
| Только длинные | 20.1% | 17.4% | 15.4% | 0.226 | 1.468 | 84.6% |
| Без ограничений | 16.9% | 13.6% | 24.0% | 3.619 | 4.048 | 10.6% |
| Короткие до 25% | 17.0% | 13.6% | 24.9% | 2.085 | 4.077 | 48.9% |

Все границы Монте-Карло лежат внутри (правее и ниже) аналитических, что подтверждает оптимальность решений Марковица. Для неограниченного режима сходимость лучшая (зазор по Шарпу всего 10.6%).

### Наиболее рискованный портфель (задача 24)

Обратная задача к Марковицу: максимизация дисперсии портфеля вместо минимизации. Доказано, что при единственном ограничении (сумма весов = 1) максимум не существует (дисперсию можно наращивать неограниченно через короткие позиции). Единственная критическая точка лагранжиана — это GMVP (минимум).

Направление максимального роста риска — первый собственный вектор проецированной ковариационной матрицы (рыночный фактор: все компоненты одного знака). Построена анти-эффективная граница (максимум дисперсии при заданной доходности) с ограничениями на леверидж от 1 до 20.

## Источники данных

- MOEX ISS API (iss.moex.com) — котировки акций и индекса IMOEX, через библиотеку apimoex
- ЦБ РФ (cbr.ru) — ключевая ставка (безрисковая ставка)

Ключи доступа не требуются.


## Особенности данных

1. Период февраль-март 2022 (приостановка торгов на Московской бирже): торги были остановлены на несколько недель. Пропуск остается в данных и описан в методологии.

2. CBOM (МКБ): торги начались 02.07.2015, поэтому общий период всех 30 акций начинается с этой даты, а не с 01.01.2015.

3. Отрицательный коэффициент Шарпа: при безрисковой ставке 16% доходность GMVP ниже безрисковой ставки во всех режимах. Это не ошибка — при ключевой ставке 16% акции объективно проигрывают депозиту на годовом горизонте.
